# Frontend (Vitrax Home)

For full project setup (frontend + backend), use the root guide:

- [`../../README.md`](../../README.md)

Quick frontend-only setup:

```bash
cd FRONTEND/vitrax-limited
npm install
cp .env.example .env
npm run dev
```

Default API base URL in `.env`:

```env
VITE_API_URL=http://localhost:5000
```
